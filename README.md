# Axer Legal Pages

Sitio web estático con las páginas legales de Axer:
- **Privacy Policy** - Política de Privacidad
- **Terms of Service** - Términos de Servicio

## Estructura

```
axer-legal/
├── index.html      # Landing page
├── privacy.html    # Política de Privacidad
├── terms.html      # Términos de Servicio
├── vercel.json     # Configuración de rutas
└── public/
    └── logo.png    # Logo de Axer
```

## URLs

Una vez desplegado en Vercel:
- `https://axer.app/` - Landing page
- `https://axer.app/privacy` - Política de Privacidad
- `https://axer.app/terms` - Términos de Servicio

## Deploy en Vercel

1. Crear repositorio en GitHub
2. Conectar con Vercel
3. Configurar dominio personalizado `axer.app`
